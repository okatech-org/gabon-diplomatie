## Resume

- **Bundle ID iOS** : `ga.diplomate.app`
- **Package Android** : `ga.diplomate.app`
- **Stack** : React Native + Expo SDK 52+ + Expo Router + NativeWind + Convex
- **Securite** : MFA obligatoire, biometrie obligatoire, sessions auditees, pas de screenshots
- **Distribution** : Google Play Store + Apple App Store + EAS Build
